Creating Simple Lean And Powerful Web Apps with Ratpack

While Grails allows for complex, multi-tiered web applications to be written. There is often a need for smaller, ligther, quick to write HTTP Applications. Ratpack fills the void in the JVM in writing simple, clean, and fast web apps. In addition, Ratpack allows much greater flexibility in what tools and library to use. While Ratpack can be written in any JVM language, it is has specific support for Groovy. This presentation will show how to write apps with Ratpack and Groovy.

Speaker: Joseph Faisal Nusairat
