Shell Processing with Groovy

Shell scripting is often hard for Groovy developers. Most of us while able to create basic BASH shell scripts have a harder time creating more advance ones. This difficulty is intensified when we are tasked to help admins with shell scripts. The solution? Use Groovy to shell script. This presentation will go over how to use Groovy to create some powerful shell scripting so that we can write shell scripts that are more robust and clean.
